# 名刺管理（Meishi）

巷の名刺管理アプリと同じ感覚で使える、個人用のシンプルなWebアプリです。
**ビルド不要・依存ゼロ**。`index.html` をブラウザで開くだけで動きます。

## 使い方

1. `meishi/index.html` をダブルクリック（またはブラウザにドラッグ）して開く
2. 初回起動時に、受け取った名刺14件がサンプルとして読み込まれます

## できること

- **一覧表示** … 会社カラーのアバター付きカードで表示
- **検索** … 名前・会社・役職・部署・メール・タグ・メモを横断検索（日英どちらでも）
- **会社で絞り込み** … 上部のチップをタップ
- **お気に入り** … ★で印を付けて「お気に入りのみ」表示
- **並び替え** … 名前 / 会社 / 最近追加
- **追加・編集・削除** … 日英両面のフィールドに対応
- **電話・メール・Webリンク** … 詳細画面からそのまま発信・送信・閲覧
- **バックアップ** … JSONで書き出し／読み込み、サンプルへのリセット

## データの保存先

データはブラウザの `localStorage`（キー: `meishi.cards.v1`）に保存されます。
別の端末やブラウザに移すときは、メニュー（⋯）から **JSONで書き出し → 読み込み** を使ってください。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の構造 |
| `app.css` | スタイル |
| `app.js` | ロジック（保存・検索・編集など） |
| `cards.seed.js` | 初期サンプルデータ（受領した名刺をデジタル化） |
