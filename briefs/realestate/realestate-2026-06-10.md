# 不動産ウォッチ 2026-06-10

## 本日該当なし

---

### 調査概要

対象エリア全域（文京区・中央区・城東・品川区一部）にて、新着・値下げ 2LDK（55㎡以上・1.2億以下・所有権・新耐震・駅徒歩7分以内）を横断スクリーニング実施。必須条件を全て満たす候補を2件抽出し、建物実成約㎡単価との比較を行ったが、いずれも発火条件（売出㎡単価 < 建物実成約㎡単価 × 95%）に達しなかった。

### 候補1：グランシティ日本橋人形町（中央区）→ 除外

| 項目 | 内容 |
|------|------|
| 所在地 | 東京都中央区日本橋人形町3丁目 |
| 価格 | 9,980万円 |
| 専有面積 | 55.06㎡ |
| 間取り | 2LDK |
| 階数 | 1階 |
| 駅徒歩 | 人形町駅（東京メトロ日比谷線・都営浅草線）徒歩4分 ✓ |
| 築年月 | 2002年6月（新耐震 ✓） |
| 土地権利 | 所有権（三菱地所分譲）✓ |

**スプレッド計算**

| 指標 | 金額 | ㎡単価 | ソース |
|------|------|--------|--------|
| 売出 | 9,980万円 / 55.06㎡ | **181.3万円/㎡** | mansion-review.jp 掲載 |
| 建物実成約（直近） | 9,300万円 / 50.98㎡（2025/01・9F・2LDK） | **182.4万円/㎡** | ieshil.com（reinfolib由来） / 住友不動産販売 |
| スプレッド | 181.3 ÷ 182.4 − 1 | **−0.6%** | 発火条件（−5%未満）未達 |

**除外理由**
現売出（1F）の181.3万円/㎡は建物成約（9F）とほぼ同水準。1階は通常9〜10階比で10〜15%低く成約する慣行があり、1Fで181万円/㎡は実質的に建物相場上限付近。相場割れは確認できない。

| 築年月 | 2010年9月（新耐震 ✓） |
| 土地権利 | 所有権（三菱地所分譲）✓ |

**スプレッド試算**

| 指標 | ㎡単価 | ソース |
|------|--------|--------|
| 売出 | 161.8万円/㎡ | 9,780万円 ÷ 60.43㎡ |
| 建物実成約（直近） | **216万円/㎡** | 2025年12月 2LDK 77.78㎡ / 1億6,840万円（ieshil.com = reinfolib由来） |
| スプレッド | **−25.1%** | 発火条件（−5%）を大幅超過 |

**本日除外の理由**
- 2025年5月24日にmansion-review.jpへ初出登録（本日時点で約13ヶ月間売れ残り）
- 今日の「新着」または「値下げ」のいずれにも該当しないことが濃厚
- 30階高層物件が建物最低価格帯（建物参考相場157〜213万円/㎡に対し161.8万円/㎡）で売れ残っている事実は、眺望遮蔽・設備不具合・旧修繕積立金不足等の隠れリスクを強く示唆

**ただし**、価格改定（値下げ）が確認された日は即日評価対象。値下げ確認後は以下の3点を最初に潰すこと：
1. 30階から実際の眺望・採光（向きと遮蔽建物確認）
2. 管理費・修繕積立金月額 + 長期修繕計画の積立充足率
3. 漏水・事故履歴（重要事項説明書で確認）
