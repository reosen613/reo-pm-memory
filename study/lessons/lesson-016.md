# Lesson 016 — EPS — Per-share earnings の 3 つの落とし穴

## 1 行で
EPS は「1 株当たり利益」だが、**どの E を使うか・何株で割るか・何と比べるか** の 3 点で簡単に歪められる。

---

## なぜ今それをやるか

Phase 1 は Graham の分析手法を地図に変えるフェーズ。Lesson 015 では投資家として有価証券報告書を読む「姿勢」を学んだ。次に必要なのは、最も頻繁に引用される指標 — EPS — の **陥穽を正確に知ること**。

PER で株を語るとき「E が信頼できるか」を問わないアナリストは多い。ファンドインタビューで「EPS を見て割安と判断した」と言っても、面接官はすぐ「その E はどれですか？ Adjusted ですか GAAP ですか 一過性除外後ですか」と返してくる。Graham が *Security Analysis* で EPS の「3 つの落とし穴」を論じたのはこのためだ。

---

## Core Content

### 落とし穴 1：**どの "E" を使うか** — 利益の質問題

会社が開示する「利益」は複数存在する。

| 利益の種類 | 内容 | リスク |
|---|---|---|
| GAAP 純利益 | 会計基準通り | 一過性損益を含む |
| Adjusted EPS | 社内定義で「調整」 | 経営者に都合よい除外 |
| Core Earnings (S&P) | R&D 控除・年金費用含む標準化版 | 一般に知名度低い |
| Non-GAAP | 株式報酬・M&A 費用など除外 | 「除外しすぎ」の乱用 |

Graham の原則：**10 年平均（あるいは景気サイクル 1 周分の平均）EPS** で PER を計算せよ。これがシラーの CAPE（Cyclically Adjusted PE Ratio）の原型。1 年の EPS でなく、景気山谷をまたいだ「正規化利益」で判断する。

> 実例：2023 年のソニーグループ（6758）は GAAP 純利益が金融部門の評価損で大きくブレるが、調整後 EPS は安定。「どちらの EPS を使うか」で PER が 15× にも 25× にも変わる。

---

### 落とし穴 2：**何株で割るか** — 希薄化の問題

EPS = 純利益 ÷ 発行済株式数

この「株式数」が一定でないことが問題になる。

**希薄化要因 3 種：**

1. **ストックオプション（SO）** — 行使価格 < 株価 のとき、財務会計上 *treasury stock method* で希薄化後株式数を増やす。SO 未行使でも希薄化 EPS に影響が出る。
2. **転換社債（CB）** — 転換時に株式増発 → 基本 EPS は下がらないが希薄化後 EPS は下がる。
3. **自社株買い（買戻し）** — 逆に株式数が減り EPS が上がる。利益が横ばいでも EPS は増加するため「成長」に見える。

**Self-check 計算：**

- 純利益 100 億円、基本株式数 1 億株 → 基本 EPS = 100 円
- SO 行使により希薄化後株式数 1.05 億株 → 希薄化 EPS = 95 円
- 自社株買いで株式数 0.90 億株 → 同じ 100 億円でも EPS = 111 円

買戻しによる EPS 押し上げは「真の利益成長」ではない。Graham は **per-share figure は便利だが、企業トータルの利益成長と切り離して判断すべき**と強調する。

---

### 落とし穴 3：**何と比べるか** — 分母（株価）の時系列問題

PER = 株価 ÷ EPS だが、どの時点の株価・どの期の EPS を使うかで全く異なる数値になる。

| 組み合わせ | 読み取れること | 盲点 |
|---|---|---|
| 現在株価 / 今期 予想 EPS | 市場コンセンサスとの比較 | 予想は外れる |
| 現在株価 / 過去 12M 実績 EPS | Trailing PER | 業績転換点を見逃す |
| 現在株価 / 10 年平均 EPS | CAPE / Graham PER | 成長企業に過小評価 |
| EV / EBIT | 資本構造を中立化 | EPS より優れる場面多い |

Graham の結論：PER は補助指標。**10 年平均 EPS を使い、PER < 15× であれば割安の目安**。ただしこれはデフォルト値であり、成長率・資本効率・業種特性で補正が必要。

---

### 3 つの落とし穴 まとめ

| 落とし穴 | 問うべき質問 | Graham の処方 |
|---|---|---|
| ① どの E か | GAAP か Adjusted か。一過性が混入していないか | 10 年 / サイクル平均利益を使う |
| ② 何株で割るか | 希薄化後 EPS を見ているか。自社株買いが押し上げていないか | 希薄化後 EPS を確認。EPS 成長 ≠ 利益成長 |
| ③ 何と比べるか | Trailing か Forward か。株価の基準時点は何か | PER < 15× は目安。EV/EBIT で資本構造を中立化 |

## Reo への実用的 application

**FIG 文脈：銀行 EPS の特殊性**

8393（りそな HD）など銀行株では、EPS 計算が特殊になる。

- 銀行は「純利益 ÷ 株式数」の単純 EPS に加え、**業務純益ベースのコア EPS** が重視される
- 持合解消・政策株売却益が一過性で GAAP 純利益を押し上げるケースが多い（→落とし穴①）
- AT1 債（劣後債）の利息はコモン EPS から控除されないが、**Distributable Earnings** の計算では考慮すべき

**Schweitzer v6 ポートフォリオ文脈：**

ショート候補 4477（BASE）や 4661（オービス）など高成長テック株の EPS は Non-GAAP Adjusted EPS が使われがち。ストックオプション費用（株式報酬）を除外した Adjusted EPS は GAAP より大きくなりがちで PER が低く見える（→落とし穴①②の複合）。面接で「なぜショートか」を問われた際、「GAAP EPS 換算だと PER が XX× で割高」と言えると説得力が増す。

**Camphora / Kaname インタビュー対策：**

エンゲージメント系ファンドは「自社株買いによる EPS 押し上げ vs 真の利益成長」を鋭く区別する。「EPS が 3 年で 1.5 倍になったが利益は 1.2 倍。差は自社株買いによる株式数減少」という分析を瞬時にできると印象が良い。

**1928（積水ハウス）長期保有ポジション文脈：**

積水ハウスは毎年安定した自社株買いを行う。「EPS 成長 6% のうち、真の利益成長は何 %か」を分解する練習に最適。
