# Lesson 013 — Convertibles — 1 + 2 features の risk-reward

## 1 行で
転換社債は「債券の床（1 feature）」と「株式のオプション（2 feature）」を一体化した楽器だが、その便利さの代償として両者の最悪面を引き受けるリスクが常に潜む。

---

## なぜ今それをやるか

前回 Lesson 012 では Graham が「避けるべきもの」としてワラント・IPO・二次発行を扱った。転換社債はそのリストの延長にある。Graham は CB を「企業側に都合よく設計され、投資家に不利なことが多い」と批判したが、同時に「適切な条件のものは企業向け社債の代替たり得る」と認めた。Reo が見るアクティビスト・エンゲージメント候補（親子上場解消・政策保有株）では CB が資本政策のツールとして使われることも多く、発行体・投資家双方の視点を理解しておく実務価値がある。

---

## Core Content

### 1. 転換社債 (CB) の定義とメカニズム

**転換社債（Convertible Bond）**とは、**「一定の条件で発行体株式に転換できる権利（転換権）を付した普通社債」**である。

| 項目 | 内容 |
|------|------|
| 額面 | 通常100円／100ドル |
| クーポン | 普通社債より低い（オプション価値が代替） |
| 転換価格（CP） | 発行時に決定。例: 発行時株価1,000円 × 転換プレミアム 20% = CP 1,200円 |
| 転換比率 | 額面 ÷ 転換価格。例: 100万円 ÷ 1,200円 = 833.3株 |
| 満期 | 典型的に3〜5年 |
| 発行体コール条項 | 株価がCPの130〜150%超で一定日数継続した場合、発行体が強制償還できる（Hard Call / Soft Call） |

---

### 2. 1 + 2 Features（Graham の分解）

Grahamは CBを以下の2つの独立した価値コンポーネントに分解した。

```
CB の価値 = ① 社債としての価値（Bond Floor）
           + ② 株式転換オプションの価値（Equity Optionality）
```

**① Bond Floor（社債フロア）**
- 転換権がないと仮定したときの純粋な社債価値
- 同格付・同満期の普通社債として計算した現在価値
- 株価がどれだけ下落しても、この価格まで損失が限定される（理論上）
- 典型例：額面100円の CB で Bond Floor ≈ 85円ならば、転換オプション価値は15円分

**② 転換オプション（Embedded Call Option）**
- 保有者は株価上昇の恩恵を受ける権利を持つ
- ブラック–ショールズで価値づけ可能（行使価格 = 転換価格 CP）
- ボラティリティが高いほど、満期が長いほどオプション価値は上昇

#### 損益プロファイルの図解（テキスト）

```
CB の価格
  ↑
  |                             /  ← 転換価値（株価に連動）
  |                            /
  |------ Bond Floor ---------/←この折れ点が転換価格付近
  |___________________________________→ 株価
      低←         CP         →高
```

株価下落局面では Bond Floor に守られ、上昇局面では株式の上値に追随する。  
これが「下値限定・上値参加」という CB の標語の根拠。

---

### 3. Graham の批判 ― なぜ CB は危険か

Graham が CB を警戒した理由は三つある。

| 批判 | 内容 |
|------|------|
| **Forced conversion リスク** | 株価が転換価格を大きく超えると発行体がコール行使 → 投資家は不利なタイミングで株式に転換させられる |
| **クーポンの貧弱さ** | 転換オプション分だけクーポンが低い。インフレや金利上昇局面では Bond Floor が崩れ、投資家は保護を失う |
| **Bond Floor の幻想** | 発行体信用が悪化すると社債価値も下落。「株も落ち、債券も落ちる」ダブル悪化が起きる（例：2020年 COVID 初期）|

---

### 4. 現代的用途：発行体 vs 投資家の思惑

**発行体（日本企業）がCBを使う理由**
- 低クーポンで資金調達コストを抑制
- 既存株主への希薄化を転換価格より株価が上昇したときにのみ発生させる「条件付き希薄化」
- ガバナンス意識の低い時代：大株主に知られにくい資本政策の抜け道として使われた歴史あり

**投資家（ヘッジファンド）がCBを使う理由**
- **CB Arbitrage**: CB を買い、発行体株をショートすることでデルタ中立・ボラ ロングを作る
- アクティビスト CB 購入: 転換して大株主化する布石
- ディストレスト場面での Bond Floor 下回り時に「ゼロコスト・ワラント」的に購入

---

### 5. バリュエーション実務チェックリスト

買い手として転換社債を評価する際に確認すべき5点：

1. **Parity（転換価値）**: 現在株価 × 転換比率。額面と比較して premia/discount を計算
2. **Bond Floor**: 市場金利と信用スプレッドで現在価値計算。Floor との乖離（CB Premium）を把握
3. **転換プレミアム**: (転換価格 − 現在株価) ÷ 現在株価。日本CBは発行時20〜30%が多い
4. **コール条項**: Hard Call Trigger（株価何%で何日継続）の確認
5. **希薄化余地**: 転換後の発行済株式数増加率。既存株主目線では希薄化コストとなる

---

### 6. 具体例：日本の CB（Lesson 012 延長）

2023年以降の日本企業は「中期経営計画の資本効率改善」文脈でCB 発行を見直す動きがあった。TSE の PBR<1 圧力を受け、ROE 向上を求められる中でも低コスト調達手段として CB を使う企業が存在する。

一方、Camphora Capital や Kaname Capital のようなエンゲージメント型ファンドは：
- **CB 発行による希薄化懸念**を議決権行使理由の一つとして使う
- **発行前に大量保有報告（5%ルール）**の下で CB を累積取得し、転換後に株主として登場するケースもある

## Reo への実用的 application

**ウォッチリスト関連**

| 銘柄 | コード | CB 観点 |
|------|--------|---------|
| 日本ペイント | 4612 ← 類似: 4204 住友ベーク | 化学セクターは設備投資資金調達でCB利用多い。低PBR修正過程でコスト構造変化に注目 |
| ANYCOLOR | 5032 (Short) | 高ボラ・グロース株。CBを発行した場合、既存株主希薄化リスクとショートテーゼ強化の可能性 |
| 8393 佐賀銀行 | Long | 銀行はCB発行しないが、**投資先企業の CB保有**(投資有価証券)が損益インパクトになる。IFRS 9で時価評価 |

**Schweitzer v6 での実務適用**

v6 ポートフォリオ内に CB 発行履歴のある企業があれば：
1. 発行時の転換価格と現在株価の関係から「転換脅威（overhang）」を計算
2. 転換後 EPS 希薄化をモデルに組み込む（diluted EPS ≠ basic EPS の差分）
3. Bull ケースでの発行体コール行使のタイムラインをトリガーとして設定

**AVI / Oasis 面接での使用シーン**

「なぜこの銘柄の資本政策に問題があるか」という問いに対し、CB を使った不透明な希薄化の歴史や、Bond Floor 崩壊のリスクを定量的に語れると深みが出る。
