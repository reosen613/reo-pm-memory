# Lesson 012 — Things to Avoid: warrants, IPOs, secondary issues

## 1 行で
Graham の "避けるもの三点セット" ── ワラント・IPO・二次発行 ── は、いずれも売り手が買い手より有利な情報を持つ非対称ゲームであり、賢明な投資家は原則として参加しない。

---

## なぜ今それをやるか

Lesson 006〜011 で「選ぶもの」（株式選択基準、割安株）を学んだ。今回はその裏面として「除外するもの」を体系化する。除外ルールを先に固めておくと、のちの DCF・マルチプル分析（Phase 3–4）でスクリーニング工数が大幅に減る。ファンドインタビューで「どんな銘柄は見ない？」と問われたとき、即答できる軸を持てる。

## Core Content

### 1. ワラント（株式購入権）

| 論点 | Graham の主張 |
|---|---|
| 定義 | 既存株主を希薄化する潜在株式。転換社債・有利発行の附属品として登場 |
| 問題点 | 内在価値なき「紙切れ」。価値は行使価格と株価の差だけで、会社が生み出す利益には紐付かない |
| 誤解 | 「レバレッジ効果」を魅力と感じるが、Graham はこれを投機の動機づけと断定 |
| 現代文脈 | ストックオプション（Lesson 024 で詳述）・MSワラント（希薄化型）が典型。**6594 日本電産** の過去ストックオプション大量付与が PBR 圧縮要因の一つ |

> **Graham の結論**: ワラントは「現金を調達する投機的手段」。発行側が有利で、投資家は常に不利なゲームを買わされている。

---

### 2. IPO（新規公開株）

Graham が IPO を嫌う理由は構造的である。

```
売り手（オーナー・幹事証券）= 高値で売りたい
買い手（投資家）          = 適正価格かどうか判断材料が乏しい
```

**4 つの情報劣位**

1. **トラックレコードが短い** ── 上場直後に開示される有価証券届出書は3〜5年分が最長
2. **比較対象が不在** ── 上場して初めて同業比較が可能になる
3. **ブックビルディング需要煽り** ── 証券会社の利益相反（引受手数料）
4. **ロックアップ解除リスク** ── 上場後 6 ヶ月で大株主が売れる

**日本の実例 ── 4477 BASE (2019 IPO)**

| 指標 | 上場時（2019/10） | 3 年後（2022/10） |
|---|---|---|
| 株価 | ¥3,800 | ¥800 |
| PER | 非収益（赤字） | 同左（赤字継続） |
| PSR | 約 30x | 約 3x |

上場時に「EC 民主化テーマ」「コロナ巣ごもり」で過熱。Graham が警告した典型例。

> **Graham の結論**: IPO は "It's Probably Overpriced"。賢明な投資家は少なくとも2〜3年間の上場実績が出るまで参加しない。例外は「壊滅的な下落後に極端に割安になった場合」のみ。

---

### 3. 二次発行（Secondary Offering / 公募増資）

公募増資（PO）は既存株主にとって **希薄化イベント** である。

**Graham が警戒する 3 シナリオ**

| パターン | 実態 | 投資家への影響 |
|---|---|---|
| 事業拡張目的 | 自己資本調達では足りないと判断した経営陣 = ROE が低い | 既存株主 EPS 希薄化 |
| 債務返済目的 | バランスシートが既に危機的状態 | 財務悪化シグナル |
| 大株主の売出し | インサイダーが高値で抜ける | 情報非対称の極致 |

**日本の公募増資慣行**

TSE 上場企業の PO は「ディスカウント率 3〜5%」で機械的に決まることが多い。しかし本質は「経営陣が今の株価を割高と思っているか？」のシグナル。2023 年以降、TSE PBR<1 圧縮圧力で自社株買いが増える一方、PO を打ち続ける会社は資本効率改善の意思が弱い疑いがある。

> **Graham の結論**: 二次発行は売り手が常に有利。「今売るべき」と判断した人物こそ最もその株の価値を知っている。

---

### 3 つの共通構造

```
ワラント / IPO / 二次発行

   共通点: 売り手側の利益 > 買い手側の利益
          投資家は構造的に不利なゲームを買わされる
          「興奮・テーマ・話題性」が判断を歪める
```

## Reo への実用的 application

**ショートポートフォリオへの応用**

Reo のショートリスト（6594/4661/5032/4911/5803/4062/4477/6027）に **4477 BASE** が入っている。Graham の IPO 嫌い論を正確に理解すると、BASE の2019年上場時の過熱が「構造的必然」だったことが見える。高成長期待・赤字・PSR 30x ── これはまさに「IPO は Probably Overpriced」の教科書例。ショート論文でこの歴史を引用できる。

**ロング候補のスクリーニングへの応用**

ロングリスト（8864/5451/1969/6440/7282/4204/1928/1605/4540/8393）の銘柄を見る際：

- 直近3年以内に **大型 PO を打った銘柄**は除外候補（希薄化 + シグナル）
- **ワラント付き転換社債** 残高が大きい銘柄は潜在希薄化を要確認（8393 りそなは CoCo 債 / AT1 に注意）

**ファンドインタビュー（Camphora/Oasis/AVI）での活用**

アクティビスト系ファンドのインタビューでは「PO を乱発する経営陣は資本効率の意識が低い」と批判できる文脈がある。Camphora・Kaname は friendly engagement だが、「持合解消 + 自社株買い」へ誘導する際に公募増資依存体質は交渉のレバーになる。
